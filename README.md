# nano-vllm-finance

A minimal, high-performance LLM inference engine tailored for financial NLP tasks.

## Overview

nano-vllm-finance is a lightweight vLLM-inspired inference framework optimized for financial document understanding, sentiment analysis, and structured data extraction.

## Project Structure

```
nano-vllm-finance/
├── docs/               # Design documents and architecture notes
├── nano_vllm/          # Core library
├── tests/              # Unit and integration tests
├── benchmarks/         # Performance benchmarks
└── examples/           # Usage examples
```

## Quick Start

```bash
pip install -e .
```
